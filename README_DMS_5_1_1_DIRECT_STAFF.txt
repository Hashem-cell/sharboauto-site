SHARBO AUTO DMS 5.1.1 — CREATION DIRECTE DES EMPLOYES

1. Le compte employé est créé immédiatement.
2. Aucun courriel d’invitation n’est nécessaire.
3. Le système génère un mot de passe temporaire modifiable.
4. Les variables Netlify SUPABASE_URL et SUPABASE_SERVICE_ROLE_KEY restent obligatoires, car la création sécurisée d’un utilisateur doit se faire côté serveur.
5. Ne placez jamais SUPABASE_SERVICE_ROLE_KEY dans un fichier JavaScript public.

Après remplacement des fichiers: Commit, Push origin, puis redeploy Netlify.

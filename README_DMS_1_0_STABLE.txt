SHARBO AUTO DMS 1.0 STABLE

1) Dans Supabase > SQL Editor, exécutez:
   SUPABASE_SHARBO_DMS_1_0_STABLE.sql

2) Le script peut être relancé après une installation partielle.
   Il ajoute notamment payments.sales_file_id avant de créer son index.

3) Après « Success », déployez le dossier du site une seule fois sur Netlify.

4) Faites Ctrl + Shift + R dans Admin V2.

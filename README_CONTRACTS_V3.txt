SHARBO AUTO — CONTRACTS ENGINE V3

1. Open Supabase > SQL Editor > New Query.
2. Copy all content from SUPABASE_CONTRACTS_V3.sql.
3. Click Run.
4. Replace your website files with this package.
5. Commit and push to GitHub/Netlify.
6. Open /admin-v2/ and use the new "Contrats" section.

Included in this version:
- Contracts table linked to customers and vehicles.
- Automatic contract numbers such as SA-2026-01001.
- New contract wizard: customer, vehicle, details, review.
- Draft, signed and cancelled statuses.
- French/English language selection.
- GST/QST, deposit, fees, discount, total and balance calculations.
- Search, filtering, editing and deletion.

PDF generation and electronic signatures are the next module.
